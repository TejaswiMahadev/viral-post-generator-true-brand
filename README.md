# Viral Post Generator

A Next.js application that generates viral Twitter threads using AI and real-time content discovery.

## Setup Instructions

1. **Clone or download the project files**

2. **Install dependencies:**
   \`\`\`bash
   npm install
   \`\`\`

3. **Set up environment variables:**
   - Copy the `.env.local` file and add your API keys
   - Get your API keys from:
     - Kimi AI: https://platform.moonshot.cn/
     - NewsAPI: https://newsapi.org/
     - SERP API: https://serpapi.com/

4. **Run the development server:**
   \`\`\`bash
   npm run dev
   \`\`\`

5. **Open your browser:**
   Navigate to `http://localhost:3000`

## Features

- Real-time content discovery from multiple sources
- AI-powered viral thread generation
- Dark theme with purple gradients
- Responsive design
- Content source tracking

## API Keys Required

- `KIMI_API_KEY`: For AI thread generation
- `NEWS_API_KEY`: For fetching latest news
- `SERP_API_KEY`: For web content search
