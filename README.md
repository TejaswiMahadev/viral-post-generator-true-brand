# ⚡ Flock (Viral Post Generator)
### *From data → insights → viral content.*

An AI-powered pipeline that fetches real-time data from multiple sources and transforms it into structured, engaging, and shareable content using Large Language Models.

---

## 🚀 Overview

This project combines **real-world data sources + AI generation** to create meaningful, high-quality outputs such as:
- Viral posts  
- Summaries  
- Learning content  
- Structured insights  

Instead of relying only on user input, the system pulls **live, relevant information** and enhances it using AI.

---

## 🔗 Data Sources

The system integrates multiple APIs to gather rich, contextual data:

- 📚 **arXiv API** → Research papers & technical content  
- 🌐 **SerpAPI** → Real-time search results & trending topics  
- 🤖 **Kimi API (Moonshot AI)** → LLM for text generation  

---

## ⚙️ How It Works

```text
User Input / Topic
        ↓
Fetch Data (arXiv / SERP APIs)
        ↓
Process & Summarize (LLM)
        ↓
Format Output (posts / insights / content)

```
